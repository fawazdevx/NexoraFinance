# Nexora Finance

Trade. Earn. Automate.

Nexora is a next-generation DeFi protocol combining a decentralized exchange with an automated yield vault, designed for retail users on Arbitrum and Robinhood.

## Features (Planned)
- DEX (swap functionality)
- Automated yield vault (ETH & USDC)
- Aave integration
- Upgradeable smart contracts
- AI-driven rebalancing strategies

Currently in development.
